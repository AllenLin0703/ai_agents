# 历史人物一生 - Coze Workflow

一个基于 Coze 平台构建的 AI 工作流，输入任意历史人物姓名，自动生成其一生的图文故事。

## 功能

输入一个历史人物名字，工作流会自动：

1. **生成生平概述** — 提炼 6 个最关键的人生阶段，以编年体时间线呈现
2. **生成绘画提示词** — 将每个人生事件转化为适合 AI 绘图的视觉描述词
3. **生成配图** — 为每个人生阶段生成历史风格插图
4. **输出完整图文故事** — 文字与图片结合，呈现人物一生轨迹

## 工作流结构

```
开始（输入人物名）
    ↓
大模型生成文案（豆包·1.8·深度思考）
    ↓
生成图片提示词（豆包·2.0·pro）
    ↓
AI 图片生成
    ↓
结束（输出图文故事）
```

## 输出示例格式

```json
{
  "title": "拿破仑的帝国之路",
  "content": "1769年，拿破仑出生于科西嘉岛。\n9岁，被送往法国本土学习军事。\n..."
}
```

## 使用方式

1. 在 [Coze](https://www.coze.cn) 平台导入本工作流文件
2. 在起始节点输入历史人物姓名（如：朱元璋、拿破仑、武则天）
3. 运行工作流，等待输出

## 技术栈

- **平台**: Coze（扣子）
- **模型**: 豆包·1.8·深度思考 / 豆包·2.0·pro
- **节点类型**: LLM 节点、图片生成节点、代码节点
